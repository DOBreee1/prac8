Суперглобальні змінні PHP

Позначення змінної Характеристика Отримане значення
$_SERVER['SERVER_PROTOCOL'] Змінна містить ім’я і версію інформаційного протоколу, який був використаний для запиту
$_SERVER['SERVER_NAME'] Ім'я хоста сервера, на якому виконується поточний скрипт
$_SERVER['REQUEST_METHOD'] Метод запиту, який був використаний для доступу до сторінки (наприклад, GET, POST)
$_SERVER['REQUEST_URI'] URI, яка була надана для доступу до сторінки
$_GET Масив змінних, переданих скрипту через параметри URL (запити GET)
$_POST Масив змінних, переданих скрипту через HTTP POST
$_FILES Масив завантажених файлів через HTTP POST
$_COOKIE Масив змінних, переданих скрипту через HTTP Cookies
$_SESSION Масив змінних сесії
$_REQUEST Масив змінних, переданих скрипту через HTTP GET, POST та Cookies
$_ENV Масив змінних середовища
